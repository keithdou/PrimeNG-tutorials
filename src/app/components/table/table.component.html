A basic table. The data is retrieved via a REST service. In the demo version, the data is delivered from the mock json server, using an example json file.

If you don't see any data, make sure you've installed the mock json server and started it from the project root: 

 npm install -g json-server
 json-server --watch mock.json

The location of the server is defined as an environment property in environemnts/environments.ts. That way, it can be easily changed for environments other than Dev.
